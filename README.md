# wuui-cli

> wuui-cli


## 添加 page 页面

```
wuui module [moduleName] page [pageName]

wuui m [moduleName] p [pageName]
```

    ├── page                    - 所有页面总目录
    │   ├── moduleName            - 某一模块目录
    │       ├── pageName          - 某一页面目录
    │         ├── page.html       - 页面html
    │         ├── page.js         - 页面级js
    │         ├── page.styl       - 页面级css
